 🚀 Awesome Dev Practice

Welcome to **Awesome Dev Practice**!

This repository is a public playground where developers can practice Git and GitHub while collaborating with others.

Whether you're a beginner making your first commit or an experienced developer helping others, everyone is welcome.

---

## 🎯 Purpose

This repository helps developers practice:

- Git Basics
- GitHub Workflow
- Commits
- Branching
- Pull Requests
- Issues
- Discussions
- Collaboration
- Code Reviews
- Open Source Contributions

---

## 🤝 How to Contribute

1. Fork this repository.
2. Clone your fork.
3. Create a new branch.
4. Make your changes.
5. Commit your work.
6. Push your branch.
7. Open a Pull Request.

---

## 🐛 Report Issues

Found a bug?

Open an Issue and describe:

- What happened
- Expected behavior
- Steps to reproduce

---

## 💡 Suggest Improvements

Have an idea?

Open an Issue or Discussion and share your suggestions.

---

## 📜 Rules

- Be respectful.
- Write meaningful commit messages.
- Keep Pull Requests focused.
- Help others learn.
- Follow GitHub Community Guidelines.

---

## 🌍 Open for Everyone

Everyone is welcome to contribute.

Happy Coding! ❤️ .
